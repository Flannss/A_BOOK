Bab 1: Pengenalan Python

Sebelum memulai sebuah Pemrograman alangkah baiknya kita mengenal apa si itu Python. Python adalah bahasa pemrograman yang sederhana dan mudah dipelajari. Cocok untuk pemula karena sintaksnya bersih dan mudah dibaca.

Bab 2: Variabel dan Tipe Data

Dalam Python, variabel tidak perlu dideklarasikan dengan tipe. Contoh:

x = 10
nama = "Andi"
pi = 3.14

Bab 3: Percabangan

Gunakan if, elif, dan else untuk logika percabangan.

if x > 5:
    print("Lebih besar dari 5")
else:
    print("5 atau kurang")
